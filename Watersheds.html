Watershed Delineation of Adirondack Park

Created using a DEM, Sinks, Fill, Con, and Flow direction tools. The work flow for this is long, albeit rewarding. The symbology of each watershed was changed to a random order of pastel colors rather than the default gradient color to better show each watershed.
